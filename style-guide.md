# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

### Primary

- Papaya Orange: #F16718

### Headings

### Body

- Family: [Barlow Medium](https://fonts.google.com/specimen/Barlow?query=barlow)
- Weights: 500, 700
- Font Sizes: 16px, 18px
- Line Heights: 26px
